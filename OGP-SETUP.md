## 🚀 OG 이미지 생성 방법

### 방법 1: HTML 템플릿 사용
1. 브라우저에서 `/public/generate-og-image.html` 열기
2. 화면을 1200x630px로 스크린샷 캡처
3. `og-image.jpg`로 저장하여 `/public/` 폴더에 업로드

### 방법 2: 디자인 툴 사용
- **Canva**: 1200x630px Facebook 커버 템플릿 사용
- **Figma**: 커스텀 1200x630px 프레임 생성
- **Photoshop**: 웹용 이미지 최적화하여 저장
